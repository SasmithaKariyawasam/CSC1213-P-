Problem Specification:- Doing arithmetic operations in java
Implementation:-
public class Ex03{
	public static void main(String args[])
	{
		int num1 = 20;
		int num2 = 5;
		
		int add = num1+num2;
		int sub = num1-num2;
		int mul = num1*num2;
		int div = num1/num2;
		int mod = num1%num2;
		
		System.out.println("Addition :"+add);
		System.out.println("Substraction :"+sub);
		System.out.println("Multiplication :"+mul);
		System.out.println("Divition :"+div);
		System.out.println("Module :"+mod);
	}
}	
			
Output:-

Addition :25
Substraction :15
Multiplication :100
Divition :4
Module :0

Conclusion:- 
You can do basic arithmetic operations using basic operators in java.
